# Rust 184 Winter Term 2026
A map of topics with links for a Rust winter term course at DePauw University

# Getting Started
### The Book
The textbook: [The Rust Programming Language Book](https://doc.rust-lang.org/book/)

This is our main resource.
### Some other resources
* [rust-lang.org](https://rust-lang.org/) The langauge website.
* [A half-hour to learn Rust](https://fasterthanli.me/articles/a-half-hour-to-learn-rust): This is an excellent summary of most of the Rust langauge feature with code examples. A great resource.
* [Installing Rust](https://rust-lang.org/tools/install/): Get the `rustup` tool.
* Get the `rust-analyzer` extension to use in VS Code. The extension developer is "rust-lang.org".

### Some Video Resources
* [UIUC Rust class Videos](https://www.youtube.com/playlist?list=PLezlz_6l_jm1rtrDXm_c_88Y_yIqAvAWA): UIUC put out some very nice <u>*and short*</u> intro videos on Rust. We will use to get up to speed on Rust Basics.

### Online Rust Environment
[Rust Playground](https://play.rust-lang.org/)
